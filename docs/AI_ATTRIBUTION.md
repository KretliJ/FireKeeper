# AI Attribution

## Development Assistance

This project was developed with the assistance of the following tools:

| AI Tool            | Version | Primary Use                                                                        |
| ------------------ | ------- | ---------------------------------------------------------------------------------- |
| **Kimi**     | 2.6     | Initial project planning, early code generation                                    |
| **DeepSeek** | Latest  | Ongoing development, debugging, refactoring, documentation, feature implementation |

## What AI Helped With

- Code structure
- Bug fixing and troubleshooting
- Documentation formatting
- UI/UX suggestions

## What Was Human-Driven

- Problem identification
- Feature prioritization
- Security considerations and disclaimer
- Critical and architectural decisions
- Security awareness
- Testing and validation
- Project direction and philosophy

## Transparency

All AI-generated code and suggestions went through human review, testing, and validation before being included in the project. The final responsibility for the code, its functionality, and its security lies with the human maintainer.

I believe in transparency and want you to know that while AI helped build FireKeeper, the vision, decisions, and accountability remain human.
